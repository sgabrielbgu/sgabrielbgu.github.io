# Gabriel Scalosub — Academic Website

[School of Electrical and Computer Engineering](https://www.bgu.ac.il/en/u/faculties/engineering/departments/school-of-electrical-and-computer-engineering/),
Ben-Gurion University of the Negev.

**Live site:** [sgabrielbgu.github.io](https://sgabrielbgu.github.io/)
